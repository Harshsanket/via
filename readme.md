# VIA — File Sharing App

**Current Version:** v0.1.0 (alpha)

**Codename:** Hatchling 🐣

**Catch the First Flight Soon**

VIA is a browser-to-browser file sharing app made for everyone — open, simple, and built to just work. Powered by Socket.IO and WebRTC, it runs across any browser and platform, delivering fast, secure peer-to-peer transfers for files of any type or size. No barriers, no complexity — sharing things should be as easy as sending a link.
 
This project is still in early alpha and evolving every day. If you run into a bug, have an idea that could make it better, or want to contribute, feel free to open an issue or reach out. Every bit of support genuinely helps move the project forward.
